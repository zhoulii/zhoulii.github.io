Mac 安装 PicGo

发表于 2019-01-15 更新于 2022-06-09 分类于 tool

01.安装步骤

下载 PicoGo 的 dmg 包，点击安装。完成后，打开 PicGo 时可能会遇到 无法打开 PicGo，因为它来自身份不明的开发者 问题。解决办法如下：

  1. 在终端执行如下命令 sudo spctl --master-disable；
  2. 打开 系统偏好设置 -> 安全性与隐私 –> 通用，允许从以下位置下载的 App 选择 任何来源；
